About Algae Growth Tracker

The Algae Growth Tracker is a web application designed to monitor and optimize the growth conditions of Spirulina algae. By tracking key parameters such as temperature, light intensity, pH level, and COâ‚‚ injection duration, users can ensure optimal growth conditions for their algae cultivation.

Features:

  • Real-time monitoring of growth parameters
  • Historical data logging and visualization
  • User-friendly settings for parameter configuration
  • Export historical data as CSV

Technologies Used:

  • HTML, CSS, and JavaScript for frontend development
  • Tailwind CSS for styling
  • Chart.js for data visualization
  • LocalStorage for data persistence
